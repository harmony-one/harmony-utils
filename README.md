## Local Development

- Clone repo\
  `git clone https://github.com/harmony-one/harmony-utils.git`

- Install top-level dependencies\
  `yarn`

- Install sub-dependencies\
  `yarn bootstrap`

- Build and watch for changes\
  `yarn start`

- Build \
  `yarn build`

- Lint \
  `yarn lint`

- Lint (fix) \
  `yarn lint:fix`
